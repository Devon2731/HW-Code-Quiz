### **It's Quiz Time!!!! JavaScript Quiz!!**
In this project, I made a simple timed JavaScript quiz built using JavaScript and browser APIS. Using browser APIs I created and engineered HTML elements, added event listeners, timers, and stored and retrieved data into/from the localStorage.
